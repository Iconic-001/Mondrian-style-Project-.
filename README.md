# Mondrian-style-Project-.
A minimalist web recreation of a Mondrian-inspired abstract composition using HTML and CSS. Built to practice CSS Grid, layout precision, and modern frontend design with a clean, responsive structure.

<img width="1512" height="865" alt="goal" src="https://github.com/user-attachments/assets/9b90b616-5ba2-4a0e-9d87-c87ffbbb1d5f" />

## ✨ Features

- Clean geometric layout
- Primary color palette
- Responsive design
- Pure HTML & CSS
- Beginner-friendly project

## 🎨 Inspiration

Inspired by the geometric compositions of Piet Mondrian and the De Stijl movement.

This project was built as a frontend practice to improve CSS Grid positioning, spacing, and layout techniques.

## 🛠️ Built With

- HTML5
- CSS3
